## :scroll: RULES
1. 문제 리스트에서 아래 개수의 문제를 푼다
    - 수요일은 공통 문제 1개, 선택 문제 1개
    - 금요일은 공통 문제 2개, 선택 문제 2개
2. `이름 / 날짜 / 문제 번호` 의 경로로 푼 문제를 업로드 한다.
   - 문제 번호는 백준은 `b`를, SWEA는 `S`를 붙여서 업로드 한다. ex) `b2577.java`  `s1234.java`
3. 문제에 대한 자유로운 질문과 회고는 코드 읽기 시간에 공유한다.
