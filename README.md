## :scroll: RULES
1. 문제는 매일 2문제씩 푼다
2. `이름 / 날짜 / 문제 번호` 의 경로로 푼 문제를 업로드 한다.
   - 문제 번호는 백준은 `b`를, SWEA는 `S`를 붙여서 업로드 한다. ex) `b2577.java`  `s1234.java`
3. 문제에 대한 질문과 회고는 코드 읽기 시간에 공유한다.
